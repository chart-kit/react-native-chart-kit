# @chart-kit/svg-renderer

SVG renderer primitives for React Native Chart Kit.

This package owns the free default renderer adapter around `react-native-svg`.

Current exports:

- `SvgSurface`
- `SvgGroup`
- `SvgPath`
- `SvgRect`
- `SvgCircle`
- `SvgText`
- `SvgLine`
- `SvgDefs`
- `SvgClipRect`
- `SvgLinearGradientDef`
- `createSvgRenderer`
- `svgRenderer`
- `createSvgTextMeasurer`
- `createSvgTestId`
